# SojournX

**Every version of you has a realm.**

SojournX is a privacy-first multi-realm identity platform where users can move between anonymous expression, social identity, private messaging, secure marketplace exchange, spiritual reflection, and self-development.

It is not just a social app.

It is a digital sanctuary.

## Core Positioning

SojournX is the encrypted multi-realm identity platform for:

- Anonymous expression
- Social networking
- Private communication
- Secure marketplace exchange
- Spiritual reflection
- Personal evolution

## Product Realms

### 1. Anonymous Realm

A private, topic-based expression space where users can post, reflect, ask questions, and engage without identity pressure.

### 2. Social Realm

Profiles, posts, stories, friends, followers, photos, reels, and visibility controls.

### 3. Private Messaging Realm

Encrypted-style private chats, secret groups, self-destructing message concepts, and voice-note-ready communication flows.

### 4. Marketplace Realm

A privacy-first marketplace for anonymous or verified listings, digital goods, services, local discovery, spiritual items, and self-development tools.

### 5. Spiritual Realm

Daily reflections, tarot/rune-inspired pulls, guided meditation prompts, emotional check-ins, symbolic visuals, and shadow-work style prompts.

### 6. Self-Development Realm

A personal growth dashboard for moods, emotional patterns, identity themes, habits, reflections, and evolution tracking.

## Important Legal / Security Notes

This prototype is a frontend scaffold only.

Before making public claims such as "end-to-end encrypted," "anonymous," "secure marketplace," or "private by design," the production app must include real backend implementation, encryption architecture, moderation systems, privacy policies, terms of service, data deletion/export controls, and legal review.

For IP protection, consult a qualified attorney regarding:

- Trademark clearance and filing for SOJOURNX
- Logo and tagline registration
- Patent review for technical identity/privacy systems
- Contractor IP assignments
- NDAs
- Copyright ownership of code, visuals, prompts, and content

## Development

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```

Then open with Expo Go, Android emulator, iOS simulator, or web depending on your Expo setup.

### Advanced Engineering Commands

```bash
# Start Expo with a cleared Metro cache
npm run start:clean

# Expo health diagnostics
npm run doctor

# Validate that all six realms exist with feature coverage
npm run audit:realms

# Create legal/security draft docs for production readiness planning
npm run governance:init

# Strict release readiness validation for legal/security scaffolding
npm run release:check

# Full preflight checks (typecheck + realm audit + release readiness)
npm run preflight

# One-command setup for governance docs + full preflight
npm run vault:ready
```
