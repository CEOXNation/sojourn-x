# Threat Model (Draft)

This document is a starter checklist for production security design.

## Assets

- User identity metadata
- Private messages and media
- Marketplace transaction data
- Moderation and audit logs

## Baseline Controls

- Authentication hardening (MFA optionality, session controls)
- Transport security and key management plan
- Abuse prevention and anomaly detection
- Data retention and deletion workflows
- Incident response runbook

## Current Status

Not security-reviewed. Do not claim cryptographic guarantees until implemented and audited.
